# Banco Nova Frontend v3.1.1

Portal bancario ficticio para portafolio, inspirado en banca chilena.

## Stack
Vite + React 18 + TailwindCSS + Ant Design + Recharts + Zustand + Axios.

## Correr
```bash
npm install
npm run dev
```

## Demo
- Login: cualquier usuario / clave (si no hay backend).
- Registro: crea usuarios mock en localStorage.

Cuando conectes Spring Boot en `http://localhost:8080/api`, el frontend usa esos endpoints.
